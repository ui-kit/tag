# tag

Styled tags for ESS

## Usage

You can import all the defaults

    %ui-kit-tag/index.ess
    
or pick and choose as skins

    html
      %ui-kit-tag/html.ess

    h1
      %ui-kit-tag/h1.ess
      
    p
      %ui-kit-tag/p.ess

## Tags

- `html`
- `body`
- `h1`, `h2`, `h3`, `h4`, `h5`, `h6`
- `p`
